# Logo and Favicon Instructions

## Required Images

Please add the following images to this directory:

### 1. Logo (`logo.jpeg`)
- **URL**: https://www.bayareacheder.org/wp-content/uploads/2025/01/cropped-WhatsApp-Image-2025-01-08-at-10.51.18.jpeg
- **Purpose**: Main logo displayed in the navigation bar and login page
- **Recommended Size**: Maximum 200px wide, transparent background preferred
- **Format**: JPEG, PNG, or SVG

### 2. Favicon (`favicon.jpg`)
- **URL**: https://subs.bayareacheder.org/favicon.jpg
- **Purpose**: Browser tab icon
- **Recommended Size**: 180x180px or larger (will be scaled automatically)
- **Format**: JPEG, PNG, or ICO

## How to Add Images

1. Download the logo from the URL above
2. Save it as `logo.jpeg` in this directory (`assets/images/`)
3. Download the favicon from the URL above
4. Save it as `favicon.jpg` in this directory (`assets/images/`)

## Where Images Are Used

- **logo.jpeg**:
  - Login/Register page header
  - Admin dashboard navigation bar
  - Substitute dashboard navigation bar

- **favicon.jpg**:
  - Browser tab icon for all pages
  - Apple touch icon for iOS devices

## Notes

The application is already configured to use these images. Once you place them in this directory, they will automatically appear on all pages.
